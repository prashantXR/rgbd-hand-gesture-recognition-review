# Vision-Based Hand Gesture Recognition with RGB-D Sensors (IJITDM 2023)

This project presents a **systematic review of vision-based hand gesture recognition using RGB-D sensors**, focusing on the methods, processing stages, and design challenges involved in gesture-based interaction systems.

📍 Published in **International Journal of Information Technology & Decision Making (IJITDM) 2023**  
📍 Authors: Prashant Rawat, Lalit Kane, Mrinal Goswami, Avani Jindal, Shriya Sehgal  

---

## 🧠 Motivation

Hand gesture recognition is an important interaction paradigm for human-computer interaction, human-robot interaction, assistive technologies, and immersive systems.

With the availability of **RGB-D sensors**, gesture recognition systems can better capture both visual appearance and depth-based spatial structure. However, performance is still affected by challenges such as cluttered backgrounds, illumination changes, hand shape variation, motion speed, scale, and self-occlusion.

This review examines how recent work has addressed these challenges and how RGB-D sensing has been used to improve gesture recognition pipelines.

---

## 🔬 Contributions

- Conducted a **systematic review** of vision-based hand gesture recognition targeting **RGB-D sensors**  
- Organized prior work using a **PRISMA-based review methodology**  
- Synthesized the major phases of gesture recognition pipelines, including:
  - image acquisition
  - preprocessing
  - tracking and segmentation
  - feature extraction
  - classification
- Highlighted key research challenges, trends, and opportunities for future gesture-based interaction systems
